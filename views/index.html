Machine Learning!

Health is a state of body. Wellness is state of being.

FitMeUp uses standard and modified machine learning algorithms to tell you the number of days (from the present day) in which you could control your life better. If you deteriorate someday and decide to keep health aside, your aim will be modified according the past performances.

FitMeUp uses Support Vector Machines (SVM) for the regression problem we have presented. Also, the learning gets personalized for each user using an Online Learning for Linear Regression. Don't worry. Just keep working on the aim. Make sure you fall in the good cluster of your competitors. The cluster thing is naive now but soon will be using the famous KMeans or the EM algorithm (we need the users). The model learns periodically with the coming users and their daily activities, presenting you with an attractive "Game" to compete with others and oneself.
